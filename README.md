# VisuAize - Algorithm Visualization Project

An interactive educational tool for visualizing data structures and algorithms, built with React and D3.js.

## 🎓 Academic Information

- **Institution**: University of Birmingham
- **Project Code**: hxc313
- **Academic Year**: 2025-26
- **Author**: HC

## 📁 Project Structure

```
hxc313/
├── VisuAize/              # Main application
│   ├── src/               # React & D3.js source code
│   │   ├── algorithms/    # Logic for 15+ algorithms
│   │   ├── components/    # UI & Visualizer components
│   │   ├── hooks/         # Custom React hooks
│   │   └── utils/         # Data parsing & helper functions
│   ├── public/            # Static assets
│   └── package.json       # Project dependencies
├── DEVELOPMENT_NOTES.md   # Detailed development history
└── README.md              # This file (project overview)
```

## 🚀 Quick Start

```bash
cd VisuAize
npm install
npm run dev
```

Visit `http://localhost:5173` to see the application.

## ✨ Current Features (Update 27)

### Data Structures
- ✅ Arrays (Bubble, Insertion, Selection, Merge Sort)
- ✅ Linked Lists (Insert, Delete, Search, Reverse)
- ✅ Binary Search Trees (Insert, Delete, Inorder Traversal)
- ✅ Stacks (Push, Pop, Peek, Clear)
- ✅ Queues (Enqueue, Dequeue, Peek, Clear)
- ✅ Heaps (Insert, Extract Max, Heapify)

### UI/UX
- ✨ Glassmorphism design with Dark/Light modes
- ⚡ Synchronized D3.js animations with ease-cubic transitions
- 🎮 Full playback controls & Keyboard shortcuts
- 🛡️ Smart input validation with warning toasts
- 📝 Code panel with real-time execution highlighting
- 📊 Execution logs & Complexity analysis

## 🛠️ Tech Stack

- **Styling**: CSS3 with gradients and animations
- **Icons**: Lucide React
